# Changelog

All notable changes to FPS Kit will be documented here.

---

## [1.0.0] — 2024

### 🎉 Initial Release

**Character Controller**
- Walk, sprint, crouch, slide system
- Coyote time + jump buffer + double jump support
- Stamina system with delayed regeneration
- Head bobbing (speed-reactive)
- Surface-aware footstep audio
- Slope handling, step height, fall multiplier

**Camera System**
- Smooth mouse look with input smoothing
- Trauma-based Perlin noise camera shake
- Pattern-based recoil system
- Q/E lean with wall detection
- Breath sway, movement tilt
- Sprint/ADS FOV transitions

**Weapon System**
- Semi-Auto, Full-Auto, Burst fire modes
- Hitscan and Projectile shoot modes
- Damage falloff + headshot multiplier
- Normal and tactical reload
- Valorant-style recoil pattern list
- Muzzle flash, shell ejection, bullet tracers, impact effects
- Overheat system with smoke effect
- Weapon sway

**Weapon Manager**
- Multi-slot inventory
- Scroll / numpad / Q quick-switch
- Pickup & drop system

**One-Click Demo Builder**
- Auto-generates scene, prefabs, materials
- Environment with cover, pillars, platforms, ramp
- Animated target dummies with health bars
- Full HUD (ammo, HP, stamina, crosshair, kill feed)
